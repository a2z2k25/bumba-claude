## üèÅ Forbidden Patterns

### LLM-Specific Mistakes

- **Never assume file contents** ‚Üí Always Read first
- **Never assume APIs exist** ‚Üí Verify with searches
- **Never contradict earlier decisions** ‚Üí Check memory
- **Never skip verification** ‚Üí Test every assertion
- **Never ignore context limits** ‚Üí Compress proactively

### Code Quality Standards

- `any` ‚Üí Specific types (prevents runtime errors)
- Missing return types ‚Üí Always specify (API contracts)
- `# type: ignore` ‚Üí Fix root cause (technical debt)
- String concatenation in SQL ‚Üí Parameterized queries
- Global mutable state ‚Üí Dependency injection
